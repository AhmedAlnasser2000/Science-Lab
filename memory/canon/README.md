# Canon Lane

`memory/canon/` is the canonical chronological memory lane.

- `verbatim_ledger.md` is the primary saved-memory timeline.
- Entries are append-only and use global `entry_id`.
- `CS` / `CANON SAVE` writes here first.

This lane is distinct from `memory/world-canon.md`:
- canon ledger = historical verbatim timeline,
- world-canon = durable invariants.
